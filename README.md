# Aplikacja Google Sheets → SQLite → Web

## Opis projektu
Aplikacja Python, która:
- Pobiera dane z Google Sheets
- Zapisuje je w bazie SQLite
- Wyświetla dane przez interfejs webowy

## Struktura projektu
```
pierwszaapka/
├── src/          # Główny kod aplikacji
├── tests/        # Testy
├── config/       # Konfiguracja
└── docs/         # Dokumentacja
```
